# Reference: PRD Reader

Before asking any questions, check if `docs/prd.md` exists in the project.

**If it exists:**
- Read the entire document
- Extract any information relevant to your current section
- Present what you found and ask the user to confirm or correct each value:
  "I got this from your PRD: '[value]' — does that look right?"
- Only ask questions for information NOT found in the PRD
- If the PRD contradicts itself, flag it to the user before proceeding

**If it doesn't exist:**
- Proceed with the normal Q&A as planned

The goal is to avoid asking the user questions their BA already answered.
