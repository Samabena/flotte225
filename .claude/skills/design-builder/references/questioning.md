# Reference: Guided Questioning

When collecting information from the user:

- Ask **one question at a time**. Never bundle multiple questions together.
- Show what you've collected so far before moving to the next question.
- If the user gives a vague answer, ask a follow-up before moving on.
- When a section is complete, summarize it and ask:
  "Does this look right? Type 'yes' to continue or tell me what to change."
- Never proceed to the next section until the user confirms the current one.
- Use plain, friendly language. Avoid jargon unless the user uses it first.
- If the user says "skip" or "not sure", write "TBD" and move on without pushing.
